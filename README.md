# Jobar Agro — Landing Page

Landing page estática para Jobar Agro. Paisagismo de alto impacto.

## Tecnologias

- HTML5 semântico
- CSS3 custom (sem frameworks)
- JavaScript vanilla
- Google Fonts (Playfair Display + DM Sans)

## Estrutura

```
index.html   — Markup completo
styles.css   — Estilos e design system
script.js    — Interações (menu, FAQ, scroll-reveal)
```

## Local

Basta abrir `index.html` no navegador.

## Deploy

Hospedado na Vercel como site estático.
